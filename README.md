## Dependencies

* gtk3 - https://gitlab.gnome.org/GNOME/gtk
* gtkmm (C++ bindings for gtk3) - https://gitlab.gnome.org/GNOME/gtkmm
* meson (meson build system) - https://github.com/mesonbuild/meson
* Ninja - https://github.com/ninja-build/ninja

## Building

```
$ meson build
$ cd build
$ ninja -C .
```
Run with:
```
$ src/gtkmm-testing
```
